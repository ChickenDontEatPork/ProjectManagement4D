# Project Management

##### This repository was create d for development the new project management in CE-KMITL

## Install

Cloning

```bash
git clone git@github.com:ChickenDontEatPork/frontend.git
```

Install Dependencies

```bash
yarn
```

## Running

Development

```bash
yarn dev
```

## Commit

```bash
yarn commit [remoteName] [branchName]
```
